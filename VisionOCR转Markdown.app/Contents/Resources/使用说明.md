# VisionOCR 转 Markdown 使用说明

这是本机离线运行的办案材料转换工具。双击应用可打开图形界面，也可把文件直接拖到应用图标上转换。

## 可处理内容

| 输入文件 | 输出 |
| --- | --- |
| PDF、PNG、JPG、TIFF、HEIC 等图片 | Apple Vision OCR 生成 Markdown |
| Word、Excel、CSV、PPT、HTML 等 | 生成 Markdown |
| Markdown、TXT | 生成 Word 文档 |
| 音频、视频 | 生成 Markdown 逐字稿 |
| 微信聊天录屏 MP4/MOV/M4V | 生成取证 PDF 初稿、截图和复核资料 |

`.doc` 老格式会先由 macOS 转成 `.docx` 后再提取内容。PDF 会自动分流：原生文本 PDF 优先走快速版面提取，扫描件和图片走本机 Apple Vision OCR。表格会尽量输出为 Markdown 表格；复杂扫描件仍需对照原件复核。

## 输出与归档

日常输出目录：

```text
~/Desktop/VisionOCR_Output/
```

每次转换还会在以下目录保存原件副本、转换结果和处理记录：

```text
~/Desktop/VisionOCR_Output/archive/时间戳_文件名/
├── input/
├── output/
└── metadata.json
```

归档用于复核来源与结果，不表示 Markdown 可以还原扫描件的原始版面。

## 使用方式

双击：

```text
/Applications/办案工具集/VisionOCR转Markdown.app
```

界面中的“转 Markdown”页面支持 PDF、图片、Word、Excel 及音视频；“Markdown → Word”页面用于输出 Word；“微信录屏取证”页面用于把聊天录屏筛成清晰、少重复、可复核的 PDF 初稿。

微信录屏取证默认输出到：

```text
~/Desktop/录屏取证输出/
```

快速初稿全程本地处理；OCR增强默认只走本机 OCR。只有在页面内勾选“手动云端整理”时，才会发送 OCR 文字、时间戳、截图编号和 SHA256，不发送录屏或截图图像，且结果需人工核实。

脚本或 AI 调用：

```bash
/Applications/办案工具集/VisionOCR转Markdown.app/Contents/MacOS/droplet "/path/to/file"
```

Agent 快速模式：

```bash
/Applications/办案工具集/VisionOCR转Markdown.app/Contents/MacOS/droplet --agent "/path/to/file"
```

底层快速入口：

```bash
visionocr-agent "/path/to/file" --output "/path/to/output.md"
```

直接打开微信录屏取证页：

```bash
/Applications/办案工具集/VisionOCR转Markdown.app/Contents/MacOS/droplet --wechat
```

常用参数：

```bash
visionocr-agent "/path/to/file.pdf" --page 1-5
visionocr-agent "/path/to/table.png" --layout tables
visionocr-agent "/path/to/file.pdf" --recognition-level accurate
```

查看本说明：

```bash
/Applications/办案工具集/VisionOCR转Markdown.app/Contents/MacOS/droplet --help
```

## 注意事项

- 本工具全部在本机处理，无需 API Key。
- 对扫描材料、图片和 OCR 结果，应对照原件人工核实后再作为正式材料使用。
- 日志位置：`/tmp/vision-ocr-app.log`。
